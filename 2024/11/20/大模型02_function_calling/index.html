大模型结构化输出

结构化输出

结构化输出（Structured Outputs）是指让 LLM 输出符合机器可解析的格式，这对于构建 AI 应用至关重要。

技术路径

JSON mode

JSON Mode 是大模型生成结构化输出的一种方式，通过特定的提示（Prompt Engineering）让大模型输出符合 JSON 格式的数据。
关键点：

  1. 结构化输出：JSON Mode 确保大模型的输出是结构化的，符合 JSON 格式，便于程序解析和处理。
  2. 提示工程：通过特定的提示词或指令，引导大模型生成 JSON 格式的输出。
  3. 应用场景：适用于需要将大模型的输出与其他系统或 API 集成的场景，如数据提取、自动化任务等。

JSON Schema

JSON Schema 是一种用于描述 JSON 数据结构的标准，可以帮助大模型生成符合特定格式的输出。通过定义 JSON Schema，可以确保大模型的输出满足业务需求，便于后续处理。

示例：使用 JSON Schema 格式化大模型的输出
1. JSON Schema 的基本结构

JSON Schema 是一个 JSON 对象，用于描述另一个 JSON 对象的结构。它通常包含以下关键字段：

  • type: 指定数据类型，如 object、string、number、boolean 等。
  • properties: 定义对象的属性及其类型。
  • required: 指定哪些属性是必需的。
